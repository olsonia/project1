How I meet the requirements:

  1. I am making this website for my cousin's event planning business, so I am thinking empathetically.
  2. All the images used in the gallery came from photos she took. I was given permission to use her logo and all photos.
  3. I do have 5 separate pages for this website.
  4. Homepage
  5. Contact Page (social media/book consultations to be able to contact her)
  6. Gallery Page (Images from different events she has done)
  7. Services Page (services she offers and details about them)
  8. About page (For people to learn about her and why she created this business)
  9. I do have a header on each page
  10. I have a navbar in my header that uses navigation links to access each page

  12. I have put all my repeated css for my navigation in its own .css file that is shared between all my files. All other .css files that are not repeated across pages are in separate stylesheets.

  14. Each page has a favicon of a shortened version of her logo with a light pink background
  15. Non-HTML resources are organized in files according to what they are. I have all images located in an images folder and css files in a CSS folder.
  16. I do not use any libraries or frameworks in my code, it is all made from CSS and HTML
  17. All my code validates without warnings or errors on Nu Validator
  18. I do think my code is neatly aligned. Some things are centered like headers of pages and the footer just because I like the way it looks like that
  19. All my elements have consistent spacing. No text bleeds against the edges of containing boxes.
  20. I do use two different font faces. The nav bar is in a serif font while the content of my page is in Century Gothic, which is sans-serif
  21. I have a form in the contact page, which would use the provided information to send an email
  22. I use an external link to link to her business instagram.
  23. I use kebab-case for all classes and ids that I created. It was consistent throughout my code
  24. My css animation is when you hover over the "book consultation now" button, it changes color from a dark pink to a light pink to grab peoples attention.
